# 📊 HR Team Attendance & Workforce Insights Dashboard

An interactive HR Analytics dashboard built in **Power BI** to analyze employee attendance, Work From Home (WFH) trends, and leave patterns. The dashboard provides HR teams with actionable insights to monitor workforce presence, identify attendance trends, and support better workforce planning.

## 📌 Project Overview

This project demonstrates the complete Power BI workflow, starting from raw Excel data to creating an interactive dashboard for HR professionals.

The dashboard helps answer important business questions such as:

- Which weekday has the highest employee attendance?
- Which day has the highest Work From Home percentage?
- Which week of the month records the highest leave?
- What is the overall workforce attendance percentage?
- Who has the highest attendance?
- Which employees frequently work from home?
- What are the Sick Leave and Menstrual Leave trends?

## 📊 DAX Measures Created

Several DAX measures were developed to calculate key HR metrics, including:

- Present Workforce %
- Work From Home %
- Sick Leave Count
- Sick Leave %
- Menstrual Leave %
- Total Working Days
- Total Present Days
- Employee Attendance %
- Employee WFH %
- Week of Month Calculation
- Attendance Rate
- WFH Rate
